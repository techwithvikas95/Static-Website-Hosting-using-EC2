# Static-Website-Hosting-using-EC2
Website hosting using AWS EC2. This project shows how to deploy a static or dynamic website on an EC2 instance with Nginx or Apache, configure security groups, connect via SSH, and manage the server in a production style setup.
# Project Overview
• Provisioned and configured an EC2 instance on AWS
• Established secure remote access using SSH
• Installed and configured Nginx as a web server
• Deployed a static website built with HTML and CSS
• Analyzed the complete HTTP request–response lifecycle

# Tech Stack
AWS, EC2, Ubuntu Linux, Nginx, HTML, CSS, SSH, HTTP

• Improved understanding of Linux environments
• Clear understanding of web server configuration
• Stronger foundation in core DevOps concepts
